# התחלה מהירה - המרת PDF לספר

## 3 צעדים פשוטים:

### 1️⃣ הפעל את השרת המקומי
```bash
start-converter-server.bat
```
**המתן עד שתראה:** `Running on http://localhost:5000`

---

### 2️⃣ פתח את פאנל הניהול
1. היכנס לאתר שלך
2. עבור ל: `/admin`
3. לחץ על טאב **"ספרים"**
4. לחץ על **"הוסף ספר"**

---

### 3️⃣ העלה את ה-PDF
1. הזן שם ספר (בעברית): `חוות דעת`
2. בחר קובץ PDF
3. לחץ **"התחל תהליך"**
4. המתן... (5-10 דקות לספר של 100 עמודים)
5. ✅ **סיום!**

---

## מה קורה מאחורי הקלעים?

```
PDF → תמונות → GitHub → MongoDB → הספר באתר ✨
```

1. ה-PDF מומר ל-תמונות JPG
2. התמונות מועלות ל-GitHub Releases
3. נוצר מיפוי (שם עברי ↔ ID אנגלי)
4. המיפוי נשמר ב-MongoDB
5. הספר מתווסף לרשימה

---

## בעיות נפוצות

### ❌ "שרת לא מחובר"
**פתרון:** הפעל את `start-converter-server.bat`

### ❌ "שגיאה בהמרה"
**פתרון:** וודא שהקובץ הוא PDF תקין

### ❌ "שגיאה בהעלאה"
**פתרון:** בדוק את `GITHUB_TOKEN` ב-`.env.local`

---

## דרישות

✅ Python 3.8+  
✅ משתני סביבה ב-`.env.local`:
```env

GITHUB_OWNER=...
GITHUB_REPO=...

```

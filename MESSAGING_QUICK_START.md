# מערכת הודעות - התחלה מהירה

## 🎯 מה זה?

מערכת הודעות מלאה המאפשרת למשתמשים לשלוח הודעות למנהלים ולקבל תגובות.

## 🚀 איך להשתמש?

### משתמש רגיל

1. **שלח הודעה למנהלים:**
   ```
   דשבורד → "שלח הודעה למנהלים" → מלא פרטים → שלח
   ```

2. **צפה בהודעות שלך:**
   ```
   דשבורד → "ההודעות שלי" → ראה הודעות ותגובות
   ```

### מנהל

1. **צפה בהודעות:**
   ```
   פאנל ניהול → טאב "הודעות"
   ```

2. **ענה להודעה:**
   ```
   לחץ "השב" → כתוב תגובה → "שלח תגובה"
   ```

3. **נהל הודעות:**
   - **סמן כנקרא** - לסימון שקראת
   - **מחק** - למחיקת הודעה

## 📊 תכונות

✅ שליחת הודעות עם נושא ותוכן  
✅ תגובות מהמנהלים  
✅ אינדיקטור הודעות חדשות בהדר  
✅ סטטוסים: חדש / נקרא / נענה  
✅ היסטוריית תגובות מלאה  
✅ רענון אוטומטי כל 30 שניות  

## 🔒 אבטחה

- משתמשים רואים רק את ההודעות שלהם
- מנהלים רואים את כל ההודעות
- כל הפעולות מאומתות

## 🎨 UI/UX

- **כפתור בדשבורד** - גישה מהירה לשליחת הודעות
- **מודל נוח** - טופס שליחה וצפייה בהודעות
- **אינדיקטור אדום** - מספר הודעות חדשות
- **צבעים ברורים** - ירוק=נענה, כחול=חדש, אפור=נקרא

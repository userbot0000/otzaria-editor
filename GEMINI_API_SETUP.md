# הגדרת Gemini AI OCR 🤖

## איך להשיג API Key של Gemini?

1. **היכנס ל-Google AI Studio:**
   - גש ל: https://aistudio.google.com/app/apikey
   - התחבר עם חשבון Google שלך

2. **צור API Key:**
   - לחץ על "Create API Key"
   - בחר פרויקט קיים או צור חדש
   - העתק את ה-API Key

3. **הוסף ל-.env.local:**
   ```bash
   GEMINI_API_KEY=AIzaSy...your-key-here
   ```

4. **הפעל מחדש את השרת:**
   ```bash
   npm run dev
   ```

## איך להשתמש?

### בעמוד העריכה:

1. **בחר שיטת OCR:**
   - **OCR** - Tesseract (מהיר, עובד אופליין, חינמי)
   - **AI** - Gemini (מדויק יותר, דורש אינטרנט ו-API key)

2. **בחר מודל Gemini** (אם בחרת AI):
   - **2.5 Pro** (מומלץ) - המודל החדש והמתקדם ביותר
   - **1.5 Flash** - מהיר וזול יותר
   - **1.5 Pro** - איזון בין מהירות ודיוק

2. **זיהוי טקסט מלא:**
   - לחץ על כפתור "זיהוי טקסט" (text_fields)
   - הטקסט יזוהה מכל העמוד

3. **זיהוי אזור נבחר:**
   - לחץ על "בחר אזור" (crop_free)
   - גרור על התמונה לבחירת אזור
   - לחץ "זהה אזור"
